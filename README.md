# Modular Rotary Workbench

A custom-designed modular rotary workbench created using aluminum extrusion, V-slot rollers, and CAD-designed turntable mechanisms.

## Overview

This project was designed as a compact rotating workbench platform for robotics, prototyping, electronics development, and mechanical testing.

The structure combines:
- Aluminum extrusion frame
- V-slot roller guidance
- Custom rotary mechanism
- CAD-designed components
- 3D printed parts
- Modular construction approach

## Features

- Smooth rotary movement using V-slot rollers
- Lightweight aluminum extrusion structure
- Compact and modular design
- Easy to assemble and modify
- Suitable for robotics and embedded system prototyping

## Design Process

The mechanism was designed in CAD and physically prototyped to validate:
- Rotary stability
- Structural rigidity
- Load distribution
- Roller alignment
- Modular assembly concept

## Hardware Used

- Aluminum extrusion
- V-slot rollers
- Bearings
- Custom plates
- 3D printed components
- Wood tabletop

## Applications

- Robotics workstation
- Electronics prototyping table
- Embedded system development
- Mechanical testing platform
- Mobile robotics experiments
